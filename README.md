# AiAeBot

### Requirements
- Python 3.5
- discord.py
- cleverbot

### Discord
- Create bot https://discordapp.com/developers/applications/me
- Invite him in the server by 
```
https://discordapp.com/oauth2/authorize?client_id=BOT_CLIENT_ID&scope=bot&permissions=0
```

### Run
```
python3 run.py
```
